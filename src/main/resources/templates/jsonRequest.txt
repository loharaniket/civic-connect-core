Auth:
1) POST http://localhost:8080/api/auth
Request: {
    "email":"sample@gmail.com",
    "password":"plainTextPass"
}
Response: return jwt token with cookie

2) POST http://localhost:8080/api/auth/refresh
Response: get cookie token and return

District Admin:
1) POST http://localhost:8080/api/dept/admin
header: Autherization -> only district admin token accept
Request:
{
    "name":"name",
    "surname": "jsurname",
    "email": "sample@gmail.com",
    "password": "plainTextPass",
    "dept_id":valid_dept_id // take from this endpoint-> http://localhost:8080/api/dept/public
}
Response:
{
    "id": db_id,
    "name":"name",
    "surname": "jsurname",
    "email": "sample@gmail.com",
    "dept_id":valid_dept_id 
}


2) POST http://localhost:8080/api/dist/admin
Request: {
    "district":"distName",
    "state":"stateName",
    "name":"name",
    "surname":"surname",
    "password":"plainPass",
    "email":"sample@gmail.com"
}
Response:
{
    "id":db_id,
    "district":"distName",
    "state":"stateName",
    "name":"name",
    "surname":"surname",
    "email":"sample@gmail.com"
}

3) POST http://localhost:8080/api/dept

{
